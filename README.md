# Pet-s-Mania
Site para a empresa "Pet's Mania".
A empresa "Pet's Mania" com CNP-J: 15.111.111/0001-15, localizada na R. Limão Azedo nº15 Bairro: Vila Esperança Turmalina - Minas Gerais, Brasil. O cliente pesquisou e descreveu como ele deseja que seja a página, sendo ela; estática, devendo conter "Home, PETs, Rações, Contato". Ele pediu para criar uma logomarca simples e conexão MySql, e as páginas em PhP. Sendo que na página PETs (Cães, Gatos, Aves).
